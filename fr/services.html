Notre mission

Il existe de nombreux sites web de jeux en ligne. Notre mission est de créer un site web avec les meilleurs jeux gratuits. Si vous programmez des jeux, nous les hébergerons et les inclurons sur notre site web.

Games

Tous nos jeux sont open source et le code source peut être trouvé sur Github.

Émulateur de console de jeux

Avec notre émulateur de console de jeux, vous pouvez émuler d'anciennes consoles de jeux.

Accès au Desktop à distance

Vous pouvez accéder à un bureau dans votre navigateur. Le bureau est hébergé dans le nuage et peut être supprimé après son utilisation. Pour l'instant, vous devez nous écrire un email pour l'utiliser.
